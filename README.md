Rx Cart eCommerce Website
Rx Cart is a modern eCommerce platform tailored for purchasing prescription drugs and healthcare products online. This repository contains the frontend codebase built with React, styled with styled-components, and integrates with various APIs for authentication and data management.

Features
User Authentication: Integration with Auth0 for secure user authentication and authorization.
Shopping Cart: Manage items in the cart, adjust quantities, and clear cart functionality.
Product Catalog: Browse a catalog of prescription drugs and healthcare products.
Order Management: Calculate order totals including subtotal, shipping fee, and total amount.
Responsive Design: Ensures optimal viewing experience across a wide range of devices.
Technologies Used
React: Frontend JavaScript library for building user interfaces.
styled-components: CSS-in-JS library for styling React components.
Auth0: Authentication and authorization platform.
React Router: Declarative routing for React applications.
REST APIs: Integration with backend APIs for fetching product data and managing cart state.
